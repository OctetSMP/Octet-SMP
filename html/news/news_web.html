Octet SMP Official Website Launched

April 19, 2026

We are excited to announce that the official Octet SMP website is now live! After weeks of development, the website is finally ready for all members. You can now access server information, rules, commands, and more — all in one place.

The website includes a member dashboard where you can see the latest news and active players. There is a complete "How to Join" page with step-by-step instructions and images. All server rules are clearly listed, and the sellable items page includes a searchable price list for everything you can sell to the server shop.

For better gameplay, we have added a page with recommended mods and tools, including Fabric, Sodium, Iris, and Complementary Shaders. There is also a full commands page with tutorials for every /trigger command, such as TPA, home, shop, and state.

Members can log in to access their account settings, where they can update their profile, change their password, and view the user agreement. The website also features a news section where all future server updates and announcements will be posted.

We hope the website makes it easier for everyone to stay connected and informed. Check it out and let us know what you think!

— Octet SMP Leadership
