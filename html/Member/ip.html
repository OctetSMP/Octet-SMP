How to Join

Follow these simple steps to join the Octet SMP server

1

Open your Minecraft Launcher, and make sure you are using version 26.1. If you don't have it, create a new installation and select 26.1.

2

Launch the game and click on the Multiplayer button from the main menu.

3

Click the Add Server button at the bottom of the multiplayer screen.

4

Enter the following details:

Server Name: Octet SMP

Server Address: octet.aternos.me

Once added, the server will appear in your multiplayer list. Select it and click Join Server to start your adventure!
